Why Teradyne?

Teradyne's customers count on us for our Near Device Under Test (DUT) technology that gives memory device manufacturers a guaranteed performance advantage.

Dynamic memory can also be referred to as volatile memory. Volatile means that the data stored inside the memory cells is temporary. Two types of dynamic memory include Synchronous Dynamic Random-Access Memory (SDRAM), which is the type of memory used to run programs in computing devices. Synchronous means the data synchronizes to the clock signal inside the device. There are also very high speed DRAM technologies called Graphics Double Data Rate (GDDR) memories designed to support high bandwidth applications for graphics, gaming, and AI.

So, using a laptop as an example, the dynamic memory is used by the Central Processing Unit (CPU) to do work. The memory needs to be fast in order for the CPU to quickly fetch and store results. Data stored in dynamic memory is volatile, it only retains data if power is applied to the device. Dynamic memory devices need to be tested using automated test equipment and the test equipment must be fast, as fast or faster than the speed of the memory. The speeds are measured in data rate.

The other type of memory device is called Storage memory or non-volatile memory. This memory has the capability to retain data while there is no power. This is the type of memory used to store everything from a mobile phone's apps, photos, and videos. Flash memory speeds are also increasing because consumers want fast access to their data.

Teradyne's Memory Test Solutions

Teradyne's Magnum memory test solutions are designed with Near DUT Test Technology. The instrument boards are within the test interface unit and test head extension which delivers the shortest distance available, in the industry, between the DUT and the test instrument. This architectural design breakthrough gives memory device manufacturers the advantage of 'at speed testing' to fully test the performance of the device.

The architectural design of the Magnum accommodates today's memory devices and is ready to test the speed of tomorrow's memory devices whether they are LPDDR, DDR, GDDR or Flash devices.
